# 🪨 Day 04 - Rock Paper Scissors

## 📌 Description

A console-based Rock Paper Scissors game built using Python. The player competes against the computer, which makes a random choice each round. The game tracks wins, losses, draws, and total matches played.

This project is part of my **100 Days of Python Challenge**.

---

## ✨ Features

- 🎮 Player vs Computer gameplay
- 🤖 Random computer choice
- ✅ Input validation
- 📊 Match statistics
- 🏆 Win/Loss/Draw counter
- 🔁 Play Again option
- ✂️ Rock, Paper & Scissors emojis

---

## 🛠️ Technologies Used

- Python 3
- `random` module

## 📚 Concepts Practiced

- Boolean Operators (`and`, `or`)

## ▶️ How to Run

1. Clone this repository.

```bash
git clone https://github.com/ShubhamBahe/100-Days-of-Python.git
```

2. Open the project folder.

```bash
cd Day-04-Rock-Paper-Scissors
```

3. Run the program.

```bash
python rock_paper_scissors.py
```
