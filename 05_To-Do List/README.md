# 📝 Day 05 - To-Do List

## 📌 Description

A simple console-based To-Do List application built with Python. It allows users to manage daily tasks using a menu-driven interface.

## ✨ Features

- ➕ Add new tasks
- 📋 View all tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 🚪 Exit the application
- ✔️ Input validation
- ⚠️ Prevents marking completed tasks multiple times

## 🛠️ Technologies Used

- Python 3

## 📚 Concepts Practiced

- Lists
- Loops (`while`)
- Conditional Statements
- User Input
- Exception Handling (`try` / `except`)
- List Methods (`append()`, `pop()`)
- `enumerate()`

## ▶️ How to Run

```bash
python todo_list.py
```
