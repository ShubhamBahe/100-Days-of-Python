# 🔐 Day 03 - Password Generator

## 📌 Description

A Python console application that generates secure random passwords based on the user's preferences. The user can choose the password length and decide whether to include uppercase letters, lowercase letters, numbers, and special symbols.

This project is part of my **100 Days of Python Challenge**.

---

## ✨ Features

- Generate passwords of any length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include special symbols
- Random password generation
- Error handling when no character type is selected

---

## 🛠️ Technologies Used

- Python 3
- `random` module

---

## 📚 Concepts Practiced

- Variables
- User Input
- Conditional Statements
- Loops
- String Manipulation

- Input Validation
