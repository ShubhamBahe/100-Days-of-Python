# 🏦 Day 06 - ATM Simulator

## 📌 Description

A console-based ATM Simulator built with Python. It allows users to log in using an account number and password, check their balance, deposit money, withdraw money, and view transaction history.

## ✨ Features

- 🔐 User Login Authentication
- 💰 Check Balance
- ➕ Deposit Money
- ➖ Withdraw Money
- 📜 Transaction History
- ❌ Invalid Input Handling
- 🔒 Account Lock After 3 Failed Attempts

## 🛠 Technologies Used

- Python 3

## 📚 Concepts Practiced

- Dictionaries
- Nested Dictionaries
- Lists
- Loops
- Conditional Statements
- Exception Handling
- Input Validation

## ▶️ How to Run

```bash
python ATM.py
```
