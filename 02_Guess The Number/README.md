# 🎯 Day 02 - Guess The Number

## Description

A Python console game where the computer generates a random number between 1 and 100, and the player has to guess it within 7 attempts.

## Features

- Random number generation
- Maximum 7 attempts
- Play Again option
- Games Played counter
- Games Won counter
- Best Score tracking

## Concepts Used

- Variables
- Loops
- Conditional Statements
- Random Module
- User Input

## Future Improvements

- Difficulty Levels
- Input Validation
- Exception Handling
